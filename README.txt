# Football Gambling Game

The Football Gambling Game is a Python-based project that simulates a simple football gambling game. Players can place bets on football matches and earn or lose virtual currency based on the outcomes of those matches. The project repository can be found [here](https://github.com/GabrielCarroll14/Football-Gambling-Game).

## Usage Instructions

Follow the steps below to use the Football Gambling Game on your system:

### Installation

1. Ensure that Python is installed on your machine. If not, download and install Python from [Python Official Website](https://www.python.org/downloads/).
2. Clone or download the project repository to your local machine using [this link](https://github.com/GabrielCarroll14/Football-Gambling-Game).

### Running the Application

#### Windows 11

1. Locate the "Football-Gambling-Game" folder on your system.
2. Open a terminal in the folder by right-clicking and selecting "Open in Terminal."
3. Run the application using the command `python main.py`.

#### Windows 10

1. Locate the "Football-Gambling-Game" folder on your system.
2. Copy the path of the folder.
3. Open the Command Prompt and navigate to the folder using the command `cd <paste path>`.
4. Run the application using the command `python main.py`.

#### macOS

1. Locate the "Football-Gambling-Game" folder on your system.
2. Right-click the folder and select "New Terminal at Folder."
3. Run the application using the command `python main.py`.

### Usage

1. Upon launching the application, follow the prompts to place bets on football matches.
2. Enter the desired amount to bet and choose the team you think will win.
3. Matches are simulated, and outcomes determine if you gain or lose virtual currency.
4. View your current balance and continue placing bets.
5. To exit the game, follow the on-screen instructions.
